# 链表做题技巧汇总

- 添加哑结点
  - 在对链表进行操作时，一种常用的技巧是添加一个哑节点（dummy node），它的next 指针指向链表的头节点。这样一来，我们就不需要对头节点进行特殊的判断了。
- 根据索引获取节点

```golang
        # 获取a索引的前一个节点
        for i:=0;i<a-1;i++{
            temp1=temp1.Next
        }

        # 获取当前b索引节点
        for j:=0;j<b;j++{
            temp2=temp2.Next
        }
```
